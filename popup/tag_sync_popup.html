This addon allows to store and restore (sync) a list of tagged messages to a file on a harddisk.

This can be used to automatically tag messages in another TB installation. A usecase would be to have identical messages tagged on a laptop and on one's dektop PC.

Another usecase is to backup the list of tagged messages.

The store button collects the messages and asks for a save location.

The restore button asks for a file location and restores tags to those messages that exist in this TB installation. This means that both TB installations can have different accounts, and restore will ony happen for those accounts that exist in both installations.
